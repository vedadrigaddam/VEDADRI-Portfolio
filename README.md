# Vedadri Gaddam — Portfolio

## 🚀 Quick Start (Free Deployment)

### Step 1 — Get Your FREE AI API Key (2 minutes)
1. Go to **https://aistudio.google.com/app/apikey**
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy the key (looks like: `AIzaSy...`)
5. Open `index.html`, find `YOUR_GEMINI_API_KEY` and replace it with your key

```html
const GEMINI_API_KEY = "AIzaSyXXXXXXXXXXXXXXXXXXXX";
```

> ✅ Google Gemini API is **completely free** — no credit card needed. 1500 requests/day free.

## 🌐 Deploy FREE on GitHub Pages (Recommended)

### Steps:
1. Go to **https://github.com** → Sign up / log in
2. Click **"New repository"** → name it `portfolio` → set to **Public** → click **Create**
3. Click **"uploading an existing file"** → drag & drop `index.html` → click **Commit changes**
4. Go to **Settings** tab → scroll down to **"Pages"**
5. Under **Source**, select **"Deploy from a branch"** → select `main` branch → click **Save**
6. Wait 1–2 minutes → your site is live at:
   ```
   https://YOUR_GITHUB_USERNAME.github.io/portfolio
   ```

## ⚡ Deploy FREE on Netlify (Easiest - Drag & Drop)

### Steps:
1. Go to **https://netlify.com** → Sign up free
2. On the dashboard, find the **"Deploy manually"** section
3. **Drag and drop your `index.html` file** directly onto the page
4. That's it! 🎉 You get a free URL like: `https://random-name-123.netlify.app`
5. You can customize the name in **Site Settings → Site name**

---

## 🔥 Deploy FREE on Vercel

### Steps:
1. Go to **https://vercel.com** → Sign up with GitHub
2. Click **"Add New Project"** → Import your GitHub repository
3. Click **Deploy** → Done!
4. Live at: `https://portfolio.vercel.app`

## 💡 Tips

- **Custom domain**: All three platforms (GitHub Pages, Netlify, Vercel) support free custom domains
- **Netlify is fastest** for beginners — pure drag and drop, live in 30 seconds
- **GitHub Pages** is best if you want version control of your portfolio
- The AI chat uses **Google Gemini 1.5 Flash** — fast, smart, and free tier is very generous
